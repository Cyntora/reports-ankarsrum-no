Rapporterna genereras av Cyntoras månatliga rapport-pipeline. Innehållet på
GitHub Pages är statiska HTML-filer renderade från GA4, Google Search Console,
Google Ads, Meta Ads, WooCommerce/Shopify, SE Ranking m.fl. via MCP-servrar.

Pipeline-koden är intern och underhålls separat i Cyntoras kontroll-repo.
